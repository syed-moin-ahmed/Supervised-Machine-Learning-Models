# 🏠 House Price Prediction Project (Machine Learning Project)
This is an ML project that predicts the price of the houses through Regression
## 📌 Description
This is a machine learning-based project that predicts house prices.

## ⚙️ Workflow
- Data Preprocessing
- Feature Encoding
- Feature Selection
- Model Training
- Model Testing & Evaluation

## 📊 Outcome
- Performance measured by MAE, MSE, RMSE, R-Squared
- Graphical Representation of Predicted vs Actual Prices

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
