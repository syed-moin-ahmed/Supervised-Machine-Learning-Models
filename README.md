# Supervised Machine Learning Models

This repository contains implementations of supervised machine learning algorithms applied to regression and classification problems.

## Scope
- Regression models  
- Classification models  
- Data preprocessing and feature engineering  
- Model training and evaluation  

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

## Structure
Each project is organized in a separate folder containing the dataset, code, and relevant documentation.

## Objective
To apply and evaluate supervised learning techniques on different datasets and problem statements.
